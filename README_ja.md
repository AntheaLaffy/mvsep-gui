# MVSEP GUI

[English](./README.md) | [中文](./README_zh.md) | [日本語](./README_ja.md)

MVSEP 音楽分離 API のためのモダンで美しい GUI ツール。

## 機能

- **モダン UI デザイン** - 複数テーマ対応の美しいインターフェース
- **テーマサポート** - 純黒、ライトパープル、亮白テーマ
- **多言語サポート** - 日本語、英語、中国語
- **アルゴリズム検索** - アルゴリズムを模糊検索
- **ドラッグ＆ドロップ** - 音声ファイルを直接アプリにドロップ
- **リアルタイム進捗** - リアルタイムステータスログ、速度とサイズ表示のプログレスバー
- **複数出力形式** - MP3、WAV、FLAC、M4A
- **出力ディレクトリ設定** - 保存先を選択可能
- **トークン自動保存** - API トークンを自動保存
- **設定キャッシュ** - 出力ディレクトリ、形式タイムアウト、アルゴリズム、オプションをキャッシュ
- **ローカル履歴** - API がなくてもローカルで分離タスクを保存
- **ネットワーク設定** - タイムアウトとプロキシ設定
- **接続テスト** - 設定で API 接続をテスト

## テーマ

| テーマ | 説明 |
|--------|------|
| 純黒 | ダークテーマ、青のアクセント |
| 亮白 | ライトクリーンスタイル |
| ライトパープル | オタク向けパープルテーマ |

## 必要環境

- Python 3.8+
- PyQt6
- mvsep-cli

## インストール

```bash
pip install PyQt6 mvsep-cli
```

## 使い方

```bash
python main.py
```

### コマンドラインオプション

```bash
python main.py --debug    # デバッグモードを有効化
```

### 環境変数

```bash
MVSEP_DEBUG=1 python main.py  # デバッグモードを有効化
```

## はじめに

1. MVSEP API トークンを入力し、「保存」をクリック
2. 音声ファイルをドラッグ＆ドロップまたはクリックして選択
3. アルゴリズムを選択（検索ボタンでフィルタリング）
4. オプションと出力形式を設定
5. 「分離開始」をクリック

## 機能の詳細

### ステータス表示
- アップロード進捗（パーセント表示）
- 待機中のキュー位置
- 処理ステータス (processing, distributing, merging)
- ダウンロード進捗、速度とサイズ表示

### ダウンロード URL
- ダウンロード完了後、ファイルの URL がログに表示
- 必要に応じてブラウザで URL をコピーしてダウンロード

### 履歴
- ローカル履歴が自動保存
- 履歴ダイアログで過去のタスクを表示
- ログタブにアプリケーションランタイムログを表示

### 設定
- **言語**: 日本語、英語、中国語の切り替え
- **テーマ**: 純黒、亮白、ライトパープルを選択
- **ミラー**: MVSEP 本家またはミラーを選択
- **タイムアウト**: API リクエストのタイムアウト設定（秒）
- **プロキシ**: システムプロキシ自動検出または手動設定
- **接続テスト**: API 接続を確認

## 言語の切り替え

1. ヘッダーの設定ボタン (⚙) をクリック
2. 希望する言語を選択
3. 「適用」をクリック

## テーマの切り替え

1. ヘッダーの設定ボタン (⚙) をクリック
2. 希望するテーマを選択
3. 「適用」をクリック

## API トークンの取得

https://mvsep.com/user-api から無料の API トークンを取得

**注意**: 初めてクリックするとメインサイトにリダイレクトされる場合があります。正しい Token 取得方法は次の通りです：

1. mvsep.com にアクセスしてログイン
2. メインサイトの右上のユーザー名をクリック
3. ドロップダウンメニューの最初の「API」をクリック
4. 表示された API キーをコピーして GUI のトークン入力欄に貼り付け

> ヒント: ブラウザにログイン履歴がある場合は、GUI の「トークン取得」リンクを直接クリックできます
