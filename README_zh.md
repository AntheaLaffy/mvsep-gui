# MVSEP GUI

[English](./README.md) | [中文](./README_zh.md) | [日本語](./README_ja.md)

一款现代化美观的 MVSEP 音乐分离 API 图形界面工具。

## 功能特性

- **现代化 UI 设计** - 美观的多主题界面
- **主题支持** - 纯黑、亮白、浅紫三种主题
- **多语言支持** - 中文、英文和日语
- **算法搜索** - 模糊搜索算法列表
- **拖放上传** - 拖放音频文件到应用
- **实时进度** - 实时状态日志、进度条显示速度和大小
- **多种输出格式** - MP3、WAV、FLAC、M4A
- **可配置输出目录** - 选择保存位置
- **自动保存令牌** - API 令牌自动保存
- **设置缓存** - 输出目录、格式、超时、算法、选项等自动缓存
- **本地历史记录** - 即使没有 API 也能本地保存分离任务
- **网络设置** - 可配置超时时间和代理支持
- **连接测试** - 在设置中测试 API 连通性

## 主题

| 主题 | 描述 |
|------|------|
| 纯黑 | 深色主题，蓝色强调色 |
| 亮白 | 浅色简洁主题 |
| 浅紫 | 二次元紫色主题 |

## 环境要求

- Python 3.8+
- PyQt6
- mvsep-cli

## 安装

```bash
pip install PyQt6 mvsep-cli
```

## 使用方法

```bash
python main.py
```

### 命令行选项

```bash
python main.py --debug    # 启用调试模式
```

### 环境变量

```bash
MVSEP_DEBUG=1 python main.py  # 启用调试模式
```

## 快速开始

1. 输入你的 MVSEP API 令牌，点击"保存"
2. 拖放音频文件或点击选择文件
3. 选择算法（使用搜索按钮筛选）
4. 配置选项和输出格式
5. 点击"开始分离"

## 功能详情

### 状态显示
- 上传进度百分比
- 等待时的队列位置
- 处理状态 (processing, distributing, merging)
- 下载进度，显示速度和大小

### 下载链接
- 下载完成后，文件中链接会显示在日志中
- 可以复制链接到浏览器下载

### 历史记录
- 本地历史自动保存
- 在历史记录对话框中查看过往任务
- 日志标签页显示应用运行日志

### 设置
- **语言**: 切换中文和英文
- **主题**: 选择纯黑、亮白或浅紫
- **镜像源**: 选择 MVSEP 主站或镜像
- **超时时间**: 设置 API 请求超时时间（秒）
- **代理**: 自动检测系统代理或手动配置
- **测试连接**: 验证 API 连通性

## 切换语言

1. 点击标题栏的设置按钮 (⚙)
2. 选择语言
3. 点击"应用"

## 切换主题

1. 点击标题栏的设置按钮 (⚙)
2. 选择主题
3. 点击"应用"

## 获取 API 令牌

从 https://mvsep.com/user-api 获取免费 API 令牌

**注意**：首次点击可能会被重定向到主站，正确的获取获取 Token 链接步骤如下：

1. 访问 mvsep.com 主站并登录账号
2. 在主站页面右上角点击您的用户名
3. 在下拉菜单中点击第一个 "API" 选项
4. 复制显示的 API Key 并粘贴到 GUI 的 Token 处

> 小提示：如果浏览器已有登录记录，可直接点击 GUI 中的获取 Token 链接
